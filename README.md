# 🚀 URL Shortener API (.NET 8)

A production-ready URL Shortener API built with ASP.NET Core.

---

## 🔥 Features

- URL shortening
- Redirection system
- Click tracking (analytics)
- Clean API design (DTO pattern)
- Validation & error handling
- Entity Framework Core (Code First)

---

## 🧪 Tech Stack

- ASP.NET Core 8
- Entity Framework Core
- SQL Server (LocalDB)
- Swagger (OpenAPI)

---

## 🚀 Future Improvements
- Redis caching
- Rate limiting
- Base62 encoding
- Authentication & Authorization
